# Cursor Landing Page Clone
This repository contains html and css code for a landing page clone of Cursor.

## Getting Started
Follow these steps to run the project locally.

1. Clone the repository:
   ```bash
   git clone  https://github.com/MEHULARORA11/Cursor_UI-UX_Clone-Web_Dev_Cohort.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cursor-landing-page
   ```
3. Open the `index.html` file in your web browser to view the landing page.
